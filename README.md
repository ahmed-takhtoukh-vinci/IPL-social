# Le projet contient une fonction TypeScript permettant de valider une adresse email
# selon les règles fournies dans l’énoncé.  
# Une adresse est considérée comme valide si elle contient exactement un caractère `@`,
# avec du texte avant et après celui-ci, sans aucun espace.  
# La partie domaine doit contenir au moins un point (`.`), et ce point ne peut pas être
# le dernier caractère de l’adresse.
# Le développement a été réalisé en suivant l’approche TDD, avec des tests écrits
# et exécutés à l’aide de Jest.
